# MAPI Demo Scripts

These scripts demonstrate MAPI's capabilities **without requiring API connections**. Perfect for hackathon presentations!

## Available Scripts

### 1. `demo_mapi_capabilities.py`
Comprehensive demonstration of all MAPI features:
- Multi-tier memory storage
- Temporal reasoning
- Smart retrieval router
- Hallucination guard (4-layer verification)
- Hybrid retrieval
- Continuous learning
- Knowledge graph extraction
- Performance metrics

**Run:**
```bash
python3 scripts/demo_mapi_capabilities.py
```

### 2. `demo_visual_comparison.py`
Side-by-side comparison of MAPI vs Standard API:
- Memory retention
- Temporal reasoning
- Hallucination prevention
- Retrieval methods
- Confidence scoring
- And more...

**Run:**
```bash
python3 scripts/demo_visual_comparison.py
```

### 3. `demo_hackathon_presentation.py`
5-7 minute presentation script for judges:
- Problem statement
- Solution overview
- Key innovations
- Live demo guide
- Track alignment
- Why MAPI wins

**Run:**
```bash
python3 scripts/demo_hackathon_presentation.py
```

### 4. `quick_demo.sh`
Run all demos in sequence:

```bash
./scripts/quick_demo.sh
```

## What These Scripts Show

✅ **No API Required** - All demonstrations use mock data
✅ **Visual Output** - Color-coded terminal output

✅ **Hackathon-Ready** - Perfect for judge presentations
